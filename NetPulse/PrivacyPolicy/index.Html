Privacy Policy

Last Updated: November 3, 2025

1. Information We Collect

NetPulse monitors your device's network connectivity status to provide real-time alerts about network type changes. We collect:

  • Network type information (Wi-Fi, 5G, 4G, 3G, etc.)
  • Signal strength levels
  • Network carrier name (optional - can be hidden via privacy settings)
  • Network transition timestamps

All data is processed locally on your device. We do NOT collect, transmit, or store any personal information on external servers.

2. How We Use Your Information

The collected network information is used solely to:

  • Display current network status in the app
  • Trigger alerts when network type changes
  • Show historical network transitions
  • Export network data logs (if you choose to)

No data is shared with third parties or transmitted to external servers.

3. Data Storage

All network monitoring data is stored locally on your device using Android's secure storage mechanisms. You can clear this data at any time by:

  • Clearing the app's data in Android Settings
  • Uninstalling the app

4. Privacy Settings

NetPulse provides privacy controls including:

  • Hide Carrier Name – Option to hide your mobile carrier's name from notifications and displays
  • Vibration Intensity – Control the strength of alert vibrations
  • Alert Customization – Choose which network transitions trigger alerts

These settings are stored locally and never transmitted.

5. Permissions

NetPulse requires the following Android permissions:

  • Network Access – To monitor network connectivity status
  • Foreground Service – To provide real-time monitoring in the background
  • Vibration – To provide haptic feedback for alerts
  • Notifications – To display network change alerts

We only use these permissions for the core functionality of the app.

6. Third-Party Services

NetPulse does not integrate with any third-party analytics, advertising, or tracking services. Your network monitoring data remains completely private.

7. Children's Privacy

NetPulse does not knowingly collect information from children under the age of 13. The app is intended for general audiences.

8. Changes to Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be reflected in the app with an updated "Last Updated" date. Continued use of the app after changes constitutes acceptance of the updated policy.
